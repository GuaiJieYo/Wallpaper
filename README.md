# 使用教程
1. 在Releases中下载最新的压缩包  
2. 在C:\Program Files (x86)\Steam\SteamApps\common\wallpaper_engine\projects\myprojects\ 解压下载的压缩包  
3. 返回wallpaper engine就能看到了（如果没看到就重启wallpaper engine）
